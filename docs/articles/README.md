# articles

Qiita や Note 向けの記事作成メモ、構成案、草稿を置く作業ディレクトリです。

## Directory

- `qiita/`
  - Qiita 向けの記事草稿、構成案、下書き
- `note/`
  - Note 向けの記事草稿、構成案、下書き

## Basic Policy

- 媒体ごとにディレクトリを分ける
- 1 記事 1 Markdown を基本とする
- 企画メモと本文草稿は分けてもよい
- 公開前の検討内容も Markdown で残す

## Suggested Naming

- `YYYYMMDD-topic-outline.md`
- `YYYYMMDD-topic-draft.md`
- `topic-memo.md`
