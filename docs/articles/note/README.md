# Note Articles

Note 向けの記事管理用 Markdown 置き場です。公開済み記事の元原稿、記事メモ、構成案、下書きを含みます。

## Intended Style

- 背景、問題意識、体験、考え方をやや厚めに書く
- 技術の話は入れてもよいが、主役は読み物としての流れに置く
- 読者が自分事として読める切り口を意識する

## Suggested Themes

- GPT-5.4 を実装担当として使った開発体験
- 自分で全部コードを書かなくてもアプリを作れるのか
- 要求を自然言語で伝える開発スタイル
- 仕様 Markdown や設計情報まで AI と一緒に整備する話

## Reader Angles

- 非エンジニア寄りだが、AI で何か作ってみたい人
- AI にどこまで実装を任せられるか知りたい人
- 個人開発や業務改善をしている人
- 自分で全部コードを書けなくても作れるのか気になる人
