Local Browser Tool

xlsx2md

English

xlsx2md is a Single-file Web App that reads Excel (`.xlsx`) files locally and extracts narrative text, tables, and images as Markdown.

  • Runs locally in the browser, with no server communication.
  • Converts all sheets automatically without manual per-sheet steps.
  • Extracts narrative text, tables, and images as Markdown.
Open xlsx2md GitHub

Purpose

  • Convert Excel workbook content into Markdown that is easier to pass to generative AI.
  • Extract narrative text, tables, and images in a form that generative AI can use more easily.
  • Process an entire workbook in one pass without manual sheet-by-sheet work.
  • Process locally without server communication.
  • Run in a web browser without installing an additional app.

How It Works

  • Read `.xlsx` files in the browser.
  • Open the file contents and read internal data.
  • Analyze sheet, image, and related workbook information.
  • Prefer cached formula values and parse formulas when needed.
  • Extract narrative text, tables, and images.
  • Detect tables and convert them into Markdown tables.
  • Extract chart configuration information only.
  • Extract shape source data as text, and output SVG when supported.
  • Assemble the workbook into Markdown output.

日本語

xlsx2md は、Excel (`.xlsx`) をローカルで読み込み、地の文と表と画像を Markdown として抽出する Single-file Web App です。

  • ブラウザ内でローカルに動作し、サーバ通信を行いません。
  • 全シートをシートごとの手作業なしで自動変換します。
  • 地の文と表と画像を Markdown として抽出できます。
xlsx2md を開く GitHub

目的

  • Excel ブックの情報を、生成AI に渡しやすい Markdown 形式に変換
  • 地の文・表・画像を、生成AIが利用しやすい形として抽出
  • シートごとの手作業なしで、ブック全体を一括処理
  • サーバと通信不要で、ローカル環境だけで処理
  • Webブラウザだけで動作し、追加アプリのインストール不要

実現方法

  • `.xlsx` ファイルをブラウザ内で読み込み
  • ファイルの中身を展開して読み取り
  • シートや画像などの情報を解析
  • 数式は保存済みの値を優先し、必要に応じて数式を解析
  • 地の文・表・画像を抽出
  • 表を検知して Markdown の表へ変換
  • グラフは設定情報のみを抽出
  • 図形は元データをテキストとして抽出し、対応できるものは SVG も出力
  • ブック全体を Markdown 形式にまとめる
