Local Browser Tool

xlsx2md

English

xlsx2md is a Single-file Web App that reads Excel (`.xlsx`) files locally and extracts narrative text, tables, and images as Markdown.

  • Runs in the browser.
  • Converts all sheets automatically.
  • Supports display / raw / both output modes.
  • Extracts narrative text, tables, and images as Markdown.

日本語

xlsx2md は、Excel (`.xlsx`) をローカルで読み込み、地の文と表と画像を Markdown として抽出する Single-file Web App です。

  • ブラウザ内で動作します。
  • 全シートを自動で変換します。
  • `display` / `raw` / `both` の出力モードを持ちます。
  • 地の文と表と画像を Markdown として抽出できます。
