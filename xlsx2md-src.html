`.xlsx` ファイルをローカルで解析し、地の文抽出・表検出・結合セル補完を行ったうえで Markdown を生成します。

入力

ローカルで処理します。サーバ送信は行いません。変換は常に全シートを対象に行います。
変換設定
表の1行目を Markdown テーブルのヘッダー行として扱います。 文字列セルの前後空白を取り除きます。 表や地の文の中で、値のない行を可能な限り除外します。 表の中で、値のない列を可能な限り除外します。
`display` は Excel の表示値寄りで出力します。

Markdown

`raw` / `both` モードでは、表示値以外の情報を含む場合があります。
解析サマリー
まだ変換していません。
表候補スコア
strong 7点以上の強い表候補 candidate 4-6点の表候補
まだ変換していません。
数式診断
resolved fallback unsupported 解決済み / フォールバック / 未対応外部参照
まだ変換していません。
