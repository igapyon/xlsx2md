Analyze a local `.xlsx` file, extract narrative text, detect tables, fill merged cells, and generate Markdown.

Input

Processing stays local in your browser. No data is sent to a server. Conversion always covers all sheets.
Conversion Settings
Use the first row of each detected table as the Markdown table header. Remove leading and trailing whitespace from text cells. Remove empty rows from detected tables and narrative blocks where possible. Remove empty columns from detected tables where possible. Include Shape Blocks and Shapes sections in the generated Markdown output.
`display` outputs values close to what Excel shows.
`github` preserves supported Excel emphasis as GitHub-compatible Markdown: bold, italic, strike, underline, and in-cell line breaks as `<br>`.
`balanced` uses both bordered candidates and value-density fallback detection.

Markdown

`raw` and `both` modes may include information beyond Excel's displayed values.
