lht-page-hero

A component for grouping a page title, subtitle, help tooltip, and menu at the top of the page.

lht-help-tooltip

A basic example using `placement` and `wide`. The help content appears on hover or focus.

Examples for single-line and multi-line input. `help-text` also works as a `title` in fallback mode.

preview + copy

lht-preview-output

An example that bundles preview and copy UI into a single tag. Use the buttons to replace the content.

mode switch

lht-input-mode-toggle

A representative example with UI switching between `file` and `source` modes.

busy state

lht-loading-overlay

Shows the overlay temporarily when the button is pressed. It also demonstrates `busy-target-id` and `disable-target-ids`.

toast

lht-toast

Lets you check the behavior of `show(message, durationMs)` in place.

alert

lht-error-alert

An example that switches among the `error`, `warning`, and `info` variants.

lht-index-card-link

A link card for top-level index pages. It generates the internal card DOM for you.

lht-page-menu

A standalone example of the top-right page menu. Press the menu button to open the panel.
